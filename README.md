# libcimbar — 彩色条码文件传输

基于 [sz3/libcimbar](https://github.com/sz3/libcimbar) (Color Icon Matrix Barcode) 概念实现的纯前端 HTML 文件传输工具。

## 功能

- **编码**：选择任意文件 → 生成彩色矩阵条码动画 → 在屏幕上播放
- **解码**：手机摄像头扫描屏幕上的动画条码 → 自动重组文件并下载

## 使用方法

### 快速启动

```bash
# 在项目目录下启动静态服务器
python3 -m http.server 8080

# 或使用 Node.js
npx serve .
```

然后在同一局域网下：
1. **电脑端**：打开 `http://<电脑IP>:8080`，切换到"编码/发送"标签页，拖入文件，点击播放
2. **手机端**：打开 `http://<电脑IP>:8080`，切换到"解码/接收"标签页，打开摄像头，对准电脑屏幕开始扫描

> ⚠️ 摄像头访问需要 HTTPS 或 localhost。在局域网中使用时，建议用 `npx serve -S` 配合自签名证书，或通过 `ngrok` 等工具创建 HTTPS 隧道。

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 网格大小 | 条码网格尺寸（越大容量越大，但对扫描精度要求更高） | 30×30 |
| 速度 | 每帧停留毫秒数 | 300ms |
| 摄像头 | 后置/前置切换 | 后置 |

## 技术原理

- 使用 4 种颜色（黑、红、绿、蓝）编码数据，每格承载 2 bits
- 30×30 网格下每帧约 155 字节有效载荷
- 外框使用青色标记色辅助定位
- 三个角有白色定位标记用于方向识别
- CRC-16 校验保证数据完整性
- 帧序列循环播放，接收端自动去重并重组文件
